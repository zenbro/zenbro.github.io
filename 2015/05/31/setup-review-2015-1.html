Setup Review (2015#1)

31 May 2015 on productivity 1 minute read

I will do these setup revisions at least two times in a year. It should help me to see all weak points in my tools and draw a roadmap for things that I want to improve in next few months. There will be some sort of to-do list for each tool.

Operating System: ArchLinux

  • Automate backups and updates
  • Add dotfiles manager (need to chose one from here)

Window Manager: i3

  • Try layout saving
  • Rethinking key mappings that conflicts with readline hotkeys
  • Add compton configuration

Taskbar: conky

  • Add on-click events (e.g. opening calendar when clicking on date)

Command Shell: zsh

  • Add more shortcuts (e.g. git commands). Great source of inspiration is here.
  • Seems like I use quite a bit of z shell features - need to dig deeper into configuration.
  • Add fzf support everywhere where it can be added

Terminal Emulator: rxvt-unicode

  • Mastering readline hotkeys after fixing i3wm key mappings.
  • Rewrite .Xdefaults with full understanding of what I'm doing.

File Manager: ranger

  • Add support for atool
  • Improve configuration

Text Editor: vim

  • Add plugins for writing
  • Add better clipboard management
  • Improve Unite configuration
  • Try to change jshint to eslint
  • Need to think of more consistent system for keymappings that starting with <leader>.
  • Learn vimscript

Main Web Browser: firefox

  • Change VimFX to vimperator and see what's happen

Secondary Web Browser (for development): chromium

Email Client: thunderbird

Instant Messaging Client: pidgin

PDF/djvu Reader: zathura

Image Viewer: feh

Audio/Video Player: vlc

BitTorrent Client: transmission

Display Manager: slim

Theme: greybird

Icons: elementary

Looks like an average ArchLinux user setup, but i love it. However, there are still a lot to improve.
